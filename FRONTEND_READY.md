## 🚀 To Start Working

### 1. Install Dependencies
```bash
cd frontend
npm install
```

### 2. Run Development Server
```bash
npm run dev
```
Then open: http://localhost:3000

### 3. Build for Production
```bash
npm run build
npm start
```
